<Section title="Getting Flag"/>
<Paragraph>
Create a dedicated directory, since we will extract the contents of that partition for immediate analysis.
</Paragraph>
<Codeblock filename="command.txt">
Command to create a new directory: mkdir directory_name
Command to extract the contents of a partition: tsk_recover -r -o partition_offset filename destination_directory
Enter that directory using cd directory_name
Use find directory_name -name pattern_name
</Codeblock>
<Paragraph>

Why am I searching for Git?
Because this challenge is titled “Git Forensics,” so I want to check the Git section.
</Paragraph>

Now that we’ve found the destination, let’s navigate to the destination directory. Use `ls -la` to display all the files and folders in the directory we’re checking.
